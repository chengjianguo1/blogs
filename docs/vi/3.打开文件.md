
# 3.打开文件

- 输入 vi 文件名 可以进入这个模式

| 命令          | 含义                                                 |
| ------------- | ---------------------------------------------------- |
| vim 文件名    | 打开文件定位到第一行                                 |
| vim + 文件名  | 打开文件定位到最后一行                               |
| vim +2 文件名 | 打开文件定位到第三行，如果没有这么多行定位到最后一行 |
